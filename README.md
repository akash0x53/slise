SLISE
----
###Simple Local Image Search Engine   
Simple image searching tool, search images based on image histogram and other features of image.

### Dependencies  
  * Python 2.7  
  * OpenCV 2.4.3 (or OpenCV 2.4.x)
  * Gtk3 for Python 
  * Glade 3.18
  * numpy 1.6.2
